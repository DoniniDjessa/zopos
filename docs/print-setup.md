# Configuration d'Impression

## Comment retirer "localhost" et les en-têtes/pieds de page lors de l'impression

Les navigateurs ajoutent automatiquement des en-têtes et pieds de page (URL, date, numéro de page). Voici comment les désactiver :

### Google Chrome / Edge
1. Ouvrez la fenêtre d'impression (Ctrl+P)
2. Cliquez sur "Plus de paramètres"
3. Décochez "En-têtes et pieds de page"
4. Enregistrez comme paramètre par défaut

### Firefox
1. Ouvrez la fenêtre d'impression (Ctrl+P)
2. Cliquez sur "Paramètres de page"
3. Décochez toutes les options sous "En-têtes et pieds de page"
4. Mettez tout sur "-- vide --"

### Solution Permanente
Pour désactiver définitivement les en-têtes/pieds de page dans Chrome :
1. Allez dans `chrome://settings/`
2. Recherchez "Impression"
3. Désactivez "En-têtes et pieds de page"

## Configuration Recommandée pour les Reçus
- Taille : A4 ou 80mm (si imprimante thermique)
- Marges : Aucune (0)
- En-têtes et pieds de page : Désactivés
- Arrière-plan graphique : Activé (pour voir les bordures)
